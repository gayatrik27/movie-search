🎬 Movie Search App

A full-stack movie search web application that allows users to search for movies and view details in a clean and modern UI.

🚀 Features

- 🔍 Search movies by name
- 🎥 Display movie posters, titles, and release year
- ⚡ Fast and responsive UI
- 🌐 Backend API integration
- ☁️ Deployed backend using Render

🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- API: OMDB API (external movie database)
- Deployment: Render
- Version Control: Git & GitHub

🔗 Project Structure

movie-app/
├── frontend/
├── backend/

📡 How It Works

1. User enters a movie name
2. Frontend sends request to backend
3. Backend fetches data from external API
4. Data is sent back and displayed on UI
